# covid19

Progressive web App, Nigeria covid 19 tracker. It can be used as a native app in your devices. 

## Technologies used 
```HTML```, ```CSS```, ```JavaScript```.

## Usage

Use it for viewing live count of Discharged, Active and Deaths of Covid-19 patients.
